Component System Consultation

A good component system provides a framework of reusable pieces that designers and developers can deploy on a site, improving the speed of content delivery. I can help to implement such a system on your website.

Design System Consultation

A design system helps designers iterate quickly in developing new content, while providing a cohesive look and feel across web properties. It also facilitates the use of a component system in the codebase, further improving delivery efficiency. Working with designers, I can facilitate the creation of such a system.

Advanced Design Implementation

Have an ambitious landing page you want built? One with tricky responsive elements, animated components, or app-like functionality? I can help with that.

Advanced Search Functionality

Using tools like Algolia and Lunr, I can help you set up advanced (and customized) search functionality on your website. Responsive full page search results for your blog or documentation are no problem.

Advanced Analytics Configuration

Need custom event tracking that goes beyond clicks or pageviews? Perhaps you need a systematized analytics framework for tracking various events on your web properties. I can provide consultation for, and implement, such a system.

A/B Testing Implementation

Using tools like optimizely, I can help you configure A/B tests that will allow you to quantify the efficacy of changes you've made on your website, and how the impact your funnel and lead generation.

Dynamic Content Creation

I can create a framework for modifying your content based on the type of user. Want users from a specific ad campaign to see different content? No problem.

Form Handling

I can set up asyncronous, custom form events to improve the user experience, and add room for additional functionality.

Multi Language Support

I can set up translation tools on your website, enabling multi-language support.

CMS Consultation

Looking for a visual interface to make your content more accessible to non-technical stakeholders? I can provide consultation on this, and implement solutions.
